Quick Summary

  • Builds historical Free Cash Flow (FCF) table
  • Forecasts revenue, margins, reinvestment, and forward FCF
  • Terminal value via Gordon Growth + Exit Multiple
  • Discounts cash flows using WACC
  • Exports formatted Excel workbook + sensitivity tables

Problem Statement

Valuation teams need fast, repeatable and auditable DCF outputs. Manual spreadsheet modeling slows execution and increases error risk. This project automates the valuation workflow to produce consistent, defensible results.

Approach

1) Historical Build

Normalize historical financials and construct a clean historical FCF series.

2) Forecast Drivers

Forecast growth, margins, reinvestment assumptions, and compute forward FCF.

3) DCF Valuation

Discount projected cash flows and estimate terminal value via two methods.

4) Output

Export a structured Excel model and include charts and sensitivity tables.
