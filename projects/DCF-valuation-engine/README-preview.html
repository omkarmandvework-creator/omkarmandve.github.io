Equity DCF Valuation Engine — README

Recruiter-friendly README view (same styling as project page).

Overview

This project implements a complete DCF valuation pipeline for NVIDIA (NVDA) using Python + Excel. It builds historical Free Cash Flow, forecasts core drivers, calculates terminal value (Gordon Growth + Exit Multiple), and exports a formatted Excel valuation model.

Key Outputs

Downloads

  • Excel model (.xlsx)
  • Notebook (.ipynb)

Screenshots

  • Assumptions (WACC + Drivers)
  • Forecast FCF + Historical FCF
  • DCF valuation summary
  • Sensitivity analysis

How to Run

  1. Download the notebook and Excel model from the project page.
  2. Open notebook in Jupyter / VS Code.
  3. Install dependencies: pandas, numpy, openpyxl.
  4. Run all cells to regenerate tables and export the Excel file (if configured).

Common GitHub Pages Fix

If downloads/images fail on the website, it’s almost always a path mismatch: GitHub Pages is case-sensitive. Keep folder names lowercase (downloads/, images/) and avoid spaces in filenames.
