Automated DCF Valuation Engine

Standardizing discounted cash flow models across companies using Python.

Problem

Traditional DCF models can be inconsistent and time‑consuming. Analysts spend hours manually forecasting financial statements, calculating weighted average cost of capital (WACC) and building Excel outputs.

Approach

I built a Python pipeline that ingests company financials, normalizes data into a consistent schema, applies forecast drivers, computes WACC via CAPM and produces scenario‑ready DCF models. The system generates multi‑sheet Excel workbooks automatically.

Impact

The automated workflow reduces model build time from days to hours and ensures repeatable, defensible assumptions. It supports scenario analysis and accelerates investment decision making.

Technologies

  • Python (pandas, numpy)
  • CAPM & WACC calculations
  • Excel automation with openpyxl
