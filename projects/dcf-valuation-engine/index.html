Quick Summary

  • Builds historical Free Cash Flow (FCF) table
  • Forecasts revenue, margins, CapEx, D&A, and FCF (multi-year)
  • Terminal value via Gordon Growth and Exit Multiple
  • Outputs equity value and value per share
  • Exports a formatted Excel valuation model

Problem Statement

Investment teams require fast, consistent, and defensible valuations. Manual DCF modeling slows analysis and introduces avoidable errors. This project automates the valuation pipeline to enable scalable, repeatable, and auditable investment analysis. Build a reusable and transparent valuation engine that converts financial inputs into a structured DCF model and produces

Approach

1) Data Preparation

Normalize historical statements and construct a clean historical FCF series.

2) Forecast Model

Forecast drivers (growth, margins, reinvestment) and compute forward FCF for multiple years.

3) Valuation

Discount future cash flows, estimate terminal value (GGM + Exit Multiple), derive enterprise value and equity value.

4) Output & Reporting

Export a formatted Excel model and include charts/screenshots for quick review.
