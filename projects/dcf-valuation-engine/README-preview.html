Equity DCF Engine — README (Website Preview)

This page mirrors the key README points in a clean, recruiter-friendly format.

Overview

This project implements a complete DCF valuation pipeline for NVIDIA (NVDA) using Python + Excel. It builds historical Free Cash Flow, forecasts drivers, computes terminal value, and exports a formatted Excel model.

Key Features

  • Historical FCF table and normalization
  • Multi-year forecasting of core drivers
  • Discounted cash flow valuation
  • Terminal value via Gordon Growth & Exit Multiple
  • Export to formatted Excel valuation workbook

Files Included

  • Notebook (.ipynb) — step-by-step valuation workflow
  • Excel model (.xlsx) — assumptions, historicals, forecast, DCF
  • Images (.png) — assumptions, forecast, valuation snapshots

How to Run

  1. Clone the repository from GitHub.
  2. Install dependencies (pandas, numpy) in a Python environment.
  3. Open and run the notebook.
  4. Excel output is generated/exported (as configured in the notebook).
